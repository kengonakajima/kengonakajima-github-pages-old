Class enchant.Core

A class which is controlling the cores main loop and scenes. There can be only one instance at a time, when the constructor is executed with an instance present, the existing instance will be overwritten. The existing instance can be accessed from enchant.Core.instance.

  • Defined in: enchant.js
  • Extends enchant.EventTarget

  • {enchant.Core} preload(assets)
    Performs a file preload. Sets files which are to be preloaded. When enchant.Core#start is called the actual loading takes place. When all files are loaded, a enchant.Event.LOAD event is dispatched from the Core object. Depending on the type of the file different objects will be created and stored in enchant.Core#assets Variable. When an image file is loaded, an enchant.Surface is created. If a sound file is loaded, an enchant.Sound object is created. Otherwise it will be accessible as a string. In addition, because this Surface object used made with enchant.Surface.load, direct object manipulation is not possible. Refer to the items of enchant.Surface.load 
      core.preload('player.gif');
  core.onload = function() {
     var sprite = new Sprite(32, 32);
     sprite.image = core.assets['player.gif']; // Access via path
     ...
  };
  core.start();
    Parameters:
    {...String} assets
    Path of images to be preloaded. Multiple settings possible.
    Returns:
    {enchant.Core} this
